# Core Capabilities

## 1. Money Transfer System
- Instant transfers
- Beneficiary management
- Quick Send (favorites)
- Bank account integration with IFSC verification

## 2. Invoice Management
- Create professional invoices, export as PDF
- Send via email/SMS and track payment status
- Custom branding for businesses

## 3. Subscription Management
- Multiple subscription tiers (daily, annual, lifetime)
- Auto-renewal, history and promo code integration

## 4. Referral & Rewards
- Unique referral code per user
- Earn rewards per referral, dedicated referral wallet
- Withdraw to bank with minimum threshold

## 5. Payment Processing
- Razorpay integration (UPI, Cards, Netbanking)
- Payment scheduling and vendor payments

## 6. Transaction Intelligence & Analytics
- Transaction history, filters, and spending analytics
- Exportable statements and visual reports
