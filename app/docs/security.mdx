# Security Features

- **Authentication & Authorization**
  - Clerk authentication, JWT token management, session timeout
- **Data Protection**
  - End-to-end encryption in transit, secure storage (Expo Secure Store)
  - PCI DSS compliance, input validation, XSS prevention
- **Transaction Security**
  - Payment tokenization, 2FA/OTP for critical transactions
  - Fraud detection and suspicious activity alerts
- **Privacy & Compliance**
  - GDPR compliance, data minimization, user consent and right to delete
